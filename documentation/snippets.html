Constructor

To create a snippet, you will need to create a new ".js" file inside the snippets folder.
Below is an example of how a default snippet file is structured: 


            var Spark = require("sparkbots")
            const Snippet = Spark.snippet("Snippet-name")
            
            Snippet.code = () => {
            
              // Execute Code Here
            
            }
            module.exports = Snippet;
