Constructor

To create an observer, you will need to create a new ".js" file inside the observers folder.
Below is an example of how a default observer file is structured: 


            var Spark = require("sparkbots")
            const Observer = Spark.observer("name")
            
            Observer.code = (client, message) => {
            
              // Execute Code Here
            
            }
            module.exports = Observer;
