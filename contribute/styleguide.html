Boxes

For messages and notes about important stuff, we structure it by having an exclamation mark followed by the word "Important". Then underneath what the actual important message is. All of this is contained in a box that is structured as follows:

                <div class="box">
                <div class="content">
                      <p>
                          <i> class="fas fa-exclamation-circle"></i> <strong>Important</strong>
                          <br>
                        This is how an <b>important message</b> should be structured.
                      </p>
                      </div>
            </div>
